\# Stretch Reminder



This project is a simple Python tool that reminds the user to stand up and stretch every 30 minutes.



The script runs continuously in the terminal and prints a reminder message at regular intervals. 

This project demonstrates how AI-assisted coding can be used to create simple automation tools.



\## How to run the script



Open the terminal in the project folder and run:



python script.py



The program will display a reminder every 30 minutes.
